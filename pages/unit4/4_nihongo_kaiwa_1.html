日本語会話　１

Nihongo Kaiwa　１



なに してる の?

Nani shiteru no?

Tá fazendo o quê?


Na nossa conversa, vamos acompanhar dois colegas de escola, num ambiente formal, mas que ao mesmo tempo acaba se tornando nossa segunda casa, a escola! Vamos aproveitar essa oportunidade para conhecer um vocabulário para a gente usar com aqueles chegados mais próximos.
Nessa conversa, Sayaka e Luis conversam sobre uma tarefa de casa.





Walk 'n' Talk


  1. さやかちゃん、 おはよう。 おはよう (ohayō) = bom dia
    Sayaka-chan, ohayō.
    Sayaka, bom dia.
  2. おはよう、 おはよう (ohayō) = bom dia ルイスくん。 なに なに　(nani) = o que してるの？ してるの　(shiteru no) = faz / está fazendo
    Ohayō, Ruisu-kun. Nani shiteru no.
    Bom dia, Luiz. tá fazendo o quê?
  3. しゅくだい．．． しゅくだい（shukudai） = tarefa de casa
    Shukudai...
    Tarefa de casa...
  4. えー？ えー　(Ee) = quee まだ まだ　(mada) = ainda やってないの？ やってない　(yatte nai) = (neg.) forma -te do verbo "yaru" - fazer
    Ee? Mada yatte nai no?
    Quee? Não tinha feito ainda?
  5. きのう、 きのう　(kinō) = ontem いろいろ いろいろ 　(iroiro) = várias coisas あって さ。 あって　(atte) = forma -te do verbo aru- ser/estar/existir/ter/acontecer
    さ　(sa) = forma casual do よ, é similar ao sabedo Português.     わすれちゃった。 わすれちゃった　(wasurechatta) = acabei esquecendo
    Kinō、 iroiro atte sa. Wasurechatta.
    Ontem, Aconteceram várias coisas sabe? Acabei esquecendo.
  6. ゲーム してた ゲーム してた　(gēmu shiteta) = estava jogando でしょう？ でしょう　(deshō) = né? / sabe?
    Gēmu shiteta deshō?
    Tava jogando, né?
  7. まぁね！ まぁね　(mā ne) = é...
    Mā ne！
    É...
  8. しゅくだい しゅくだい　(shukudai) = tarefa de casa は 今日 今日　(kyō) = hoje ださない、 しださない　(dasanaito) = se não entregar 先生 先生　(sensē) = professor おこる おこる　(okuru) = ficar bravo よ。 よ　(yo) = informal, é similar ao sabedo Português.
    Shukudai wa kyō dasanaito, sensē okuru. yo.
    Se você não entregar a tarefa hoje, o professor vai ficar bravo
  9. だから だから　(dakara) = por isso 今 今　(ima) = agora やってん やってん　(yatten) = contração de yatte + iru + n - estar fazendo じゃん。 じゃん　(jan) = contração de janai - né?
    Dakara ima yatten jan.
    Por isso que eu tô fazendo agora, né!?
  10. そっかー。 そっかー　(sokkā) = ah, entendi じゃー、 じゃー　(jā) = então がんばって ねー！ がんばって ねー　(ganbatte nē) = boa sorte
    Sokkā. Jā, ganbatte nē!
    Ah, entendi. Boa sorte, então.
